# MAAI Bootcamp

## Overview

This repository contains the supporting materials for the bootcamp for the Master of Applied AI. This bootcamp was designed for students in their initial week within the program. It is intended to review important concepts, tools and techniques.

## Content and Goals per Day

|**Day** | **Title** | **Goals**|
|---|---|---|
| **[Day 1](d1/README.md)**|Introduction, Setup and ML Review|<p>I want to be able to work with python environments</p><p>I want to know how to pre-process a dataset</p><p>I want to know how to do a train-test-split</p><p>I want to know how to train and test a machine learning algorithm</p>
| **[Day 2](d2/README.md)**|Evaluation of ML|<p>I want to know how to evaluate an algorithm</p><p>I want to know the ML-lifecycle</p>|
| **[Day 3](d3/README.md)**|Basic Deep Learning and AI history|<p>I want to know the basics of neural networks</p><p>I want to be able to build a sequential model</p><p>I want to understand activation functions</p><p>I want to be able to preprocess data</p><p>I want to get an overview of the history of artificial intelligence|
| **[Day 4](d4/README.md)**|Computer Vision, GIT basics, and Preparations for the Hackathon|<p>I want to know what convolutional NN are</p><p>I want to be able to build a convolutional neural network</p><p>I want to be able to code a convolutional neural network</p><p>I want to be able to pre-process images</p><p>I want to be able to implement posthoc methods for computer vision</p><p>I want to know the basics of git</p><p>I want to be prepared for the Hackaton<p><p>I want to get introduced to paper prototyping</p>|
| **Day 5**|Hackaton|<p>I want to be able to use Flask</p><p>I want to be able to use Python websockets</p><p>I want to reflect on the past week and devise learning goals for myself</p><p>I want to be able to test a prototype</p>|
